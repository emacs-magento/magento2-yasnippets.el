** Overview

This package provides a collection of Magento 2 snippets for yasnippet package.

** Install
*** Manual install
#+BEGIN_SRC emacs-lisp
(add-to-list 'load-path "/path/to/magento2-yasnippets")

(require 'yasnippet)
(require 'magento2-yasnippets)
#+END_SRC
